# Architecture Decision Record (plantilla)

Solo **plantilla** para archivos `ADR-XXX-<slug>.md` bajo `docs/adr/`.

Al generar el archivo en el repositorio del producto, **no** copiar este documento tal cual: crear solo el ADR con la plantilla siguiente.

**Contenido por sección:** incluir **solo** lo **acordado** o **inferible con criterio** del mensaje del usuario y del contexto (sin inventar decisores, alternativas ni consecuencias). **No** supuestos presentados como hechos, **no** texto placeholder genérico ni listas «de relleno». Datos faltantes o decisiones pendientes → sección **Referencias**, notas al pie de metadatos o apartado explícito en **Contexto**/**Consecuencias** según aplique, sin disimular lagunas.
